# Dashboard de Toma de Decisiones — E-Commerce Brasil

**Asignatura:** BIY7121 Minería de Datos  
**Actividad:** Guía 3.6.2 — Integración con Toma de Decisiones  
**Dataset:** Brazilian E-Commerce Public Dataset (2016–2018)

---

## Archivos

| Archivo | Descripción |
|---|---|
| `dashboard_toma_decisiones.html` | Dashboard interactivo HTML con todos los insights y gráficos. Abrir directamente en el navegador (no requiere servidor). |
| `Guia toma de desiciones MEJORADA.ipynb` | Notebook Python con el análisis completo: RFM, Apriori, Cohortes, Recomendaciones y Simulación. |

## Cómo ejecutar el notebook

1. Instalar dependencias:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn mlxtend
   ```
2. Colocar `Brazilian E-Commerce Public Dataset.csv` en el mismo directorio.
3. Ejecutar todas las celdas en Jupyter o Google Colab.

## Contenido del análisis

- **Sección 1** — Exploración geográfica y temporal  
- **Sección 2** — Segmentación RFM con K-Means (4 clusters)  
- **Sección 3** — Reglas de Asociación con Apriori  
- **Sección 4** — Motor de Recomendaciones Post-Compra  
- **Sección 5** — Análisis de Cohortes y Factores de Abandono  
- **Sección 6** — Insights, Diseño de Campaña y Simulación de Impacto
